👑 Crown Clothing

Crown Clothing is a modern e-commerce web application built using React.
It allows users to browse clothing collections, securely sign in, manage their cart, and complete payments.

🚀 Features

🔐 User authentication using Firebase

🔑 Sign in with Google

🛡️ Secure login and signup system

🛒 Add to cart functionality

📦 Product collections and categories

💳 Payment integration (Stripe)

📱 Fully responsive UI

⚡ State management for cart and user data

🛠️ Tech Stack

React

Firebase Authentication

Google Sign-In API

Stripe (Payment Integration)

JavaScript (ES6+)

CSS / SCSS

📦 Functionality

Users can create an account or sign in using Google

Products are fetched from a database

Users can add/remove items from the cart

Cart updates dynamically in real-time

Users can proceed to secure checkout and payment

📁 Project Setup

Clone the repository:

git clone https://github.com/shredhaya-goel/crown-clothing.git

Install dependencies:

npm install

Run the project:

npm start
