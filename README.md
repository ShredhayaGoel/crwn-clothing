Crown Clothing 👑
Crown Clothing is a modern e-commerce web application built with React.
It allows users to browse clothing collections, sign in securely, add items to the cart, and make payments.

🚀 Features
User authentication using Firebase
Sign in with Google
Secure login and signup system
Add to cart functionality
Product collections and categories
Payment integration
Responsive UI built with React
State management for cart and user data
🛠 Tech Stack
React
Firebase Authentication
Google Sign-In API
Stripe / Payment API
JavaScript (ES6+)
CSS / SCSS
📦 Functionality
Users can create an account or sign in using Google.
Products are fetched from a database.
Users can add or remove products from the cart.
Cart updates dynamically.
Users can proceed to secure payment.
🎯 Purpose
This project was built as part of a React course to practice building a full e-commerce application with authentication, APIs, and payment features.
